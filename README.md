# Sonar Rock vs Mine Classification

## Overview

This project uses Machine Learning to classify underwater objects as:

* Rock 
* Mine

based on sonar signal data.

---

## Dataset

* 208 samples
* 60 numerical features
* Target:

  * R → Rock (0)
  * M → Mine (1)

---

## Steps Performed

* Data Loading
* Data Preprocessing
* Feature Scaling
* Model Training (SVM)
* Model Evaluation
* Prediction System

---

## Model Used

Support Vector Machine (SVC)

---

## Results

* Good accuracy on test data
* Works well for high-dimensional data

---

##  How to Run

```bash
git clone https://github.com/yourusername/sonar-rock-vs-mine.git
```
